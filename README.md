# YoutubeLapTimecodes
## Usage: 
- Open **YoutubeTimecode.swift**
- Edit the **offsetInSeconds** on your offset
- Fill the **laps** in the format:
```swift
let laps: [Float] = [
    24.879,
    22.479,
    22.801,
]
```
- In terminal: **swift YoutubeTimecode.swift run**
- The script will create a file **timecodes.txt**
